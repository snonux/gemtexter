Bright Spark

A newspaper layout with complementary colors

Welcome to Bright Spark

This theme features a carefully crafted newspaper layout with a dark complementary color scheme. The typography combines serif fonts for headings with sans-serif for body text.

Typography Showcase

The oxygen font family provides excellent readability for body text, while roboto-slab adds character to headings. Code blocks use intelone-mono for clarity.

Color Palette

Primary: #4d8596 | Secondary: #8f6254 | Accent: #49646c

Interactive Elements

Links like this example and longer text links that demonstrate the theme's navigation style use the secondary color.

Code Examples

Inline code like theme.generate() and larger blocks:

// Theme configuration
const theme = {
    name: "bright_spark",
    layout: "newspaper",
    colors: {
        primary: "#4d8596",
        secondary: "#8f6254",
        accent: "#49646c"
    },
    fonts: {
        heading: "roboto-slab",
        body: "oxygen",
        code: "intelone-mono"
    }
};

Content Structure

  • Clean, readable typography with 17px base font size
