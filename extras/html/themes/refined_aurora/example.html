Refined Aurora

A hero layout with split complementary colors

Welcome to Refined Aurora

This theme features a carefully crafted hero layout with a light split complementary color scheme. The typography combines display fonts for headings with sans-serif for body text.

Typography Showcase

The Lato font family provides excellent readability for body text, while Abril Fatface adds character to headings. Code blocks use consola-mono for clarity.

Color Palette

Primary: #3186b1 | Secondary: #a43e4d | Accent: #a4913e

Interactive Elements

// Theme configuration

    name: "refined_aurora",
    layout: "hero",
    colors: {
        primary: "#3186b1",
        secondary: "#a43e4d",
        accent: "#a4913e"

    fonts: {
        heading: "Abril_Fatface",
        body: "Lato",
        code: "consola-mono"
    }
};
