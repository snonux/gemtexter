Soft Horizon

A card layout with triadic colors

Welcome to Soft Horizon

This theme features a carefully crafted card layout with a dark triadic color scheme. The typography combines display fonts for headings with serif for body text.

Typography Showcase

The Merriweather font family provides excellent readability for body text, while Abril Fatface adds character to headings. Code blocks use hack for clarity.

Color Palette

Primary: #c84965 | Secondary: #65c849 | Accent: #4965c8

Code Examples

Inline code like theme.generate() and larger blocks:

// Theme configuration
const theme = {
    name: "soft_horizon",
    layout: "card",
    colors: {
        primary: "#c84965",
        secondary: "#65c849",
        accent: "#4965c8"
    },
    fonts: {
        heading: "Abril_Fatface",
        body: "Merriweather",
        code: "hack"
    }
};
