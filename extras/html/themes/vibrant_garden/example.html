Vibrant Garden

A overlap layout with monochromatic colors

Welcome to Vibrant Garden

This theme features a carefully crafted overlap layout with a light monochromatic color scheme. The typography combines display fonts for headings with serif for body text.

Typography Showcase

The Merriweather font family provides excellent readability for body text, while repetition-scrolling adds character to headings. Code blocks use intelone-mono for clarity.

Color Palette

Primary: #bd7f21 | Secondary: #795622 | Accent: #b88f52

Interactive Elements

Links like this example and longer text links that demonstrate the theme's navigation style use the secondary color.

Code Examples

Inline code like theme.generate() and larger blocks:

// Theme configuration
const theme = {
    name: "vibrant_garden",
    layout: "overlap",
    colors: {
        primary: "#bd7f21",
        secondary: "#795622",
        accent: "#b88f52"
    },
    fonts: {
        heading: "repetition-scrolling",
        body: "Merriweather",
        code: "intelone-mono"
    }
};

Content Structure

  • Clean, readable typography with 14px base font size
  • Heading scale ratio of 1.42x for visual hierarchy
  • Line height of 1.61 for comfortable reading
  • Overlap layout optimized for content flow
