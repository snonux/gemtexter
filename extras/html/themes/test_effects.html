RETRO THEME EFFECTS TEST

If you can see the visual effects below, they are working correctly in your browser.

Each effect should be clearly visible and animated.

Scanlines

You should see horizontal lines moving slowly downward.

This simulates old CRT monitor scanlines.

CRT Effect

You should see fine horizontal lines and darkened edges.

This creates a curved CRT monitor appearance.

Grid

You should see a green grid pattern overlay.

This creates a retro computer grid aesthetic.

Dots

You should see pulsing dot patterns.

This simulates old dot-matrix displays.

Terminal

You should see text glow and a scanning line.

This recreates classic terminal aesthetics.

No Effect

This box has no special effects for comparison.

Yellow border indicates no retro effects.

Testing Individual Themes

To test a specific theme, open its example.html file directly:

e.g., phosphor_stream/example.html

The effects should now be clearly visible in Firefox and other modern browsers.
