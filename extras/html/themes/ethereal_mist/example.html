Welcome to Ethereal Mist

This theme features a carefully crafted hero layout with a light split complementary color scheme. The typography combines display fonts for headings with sans-serif for body text.

Typography Showcase

The oxygen font family provides excellent readability for body text, while pixelon adds character to headings. Code blocks use hack for clarity.

Color Palette

Primary: #9b3f1e | Secondary: #2b8e79 | Accent: #2b448e
