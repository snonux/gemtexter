Sharp Canyon

A retro layout with monochromatic colors

Welcome to Sharp Canyon

This theme features a carefully crafted retro layout with a dark monochromatic color scheme. The typography combines serif fonts for headings with sans-serif for body text.

Typography Showcase

The oxygen font family provides excellent readability for body text, while roboto-slab adds character to headings. Code blocks use intelone-mono for clarity.

Color Palette

Primary: #b98f33 | Secondary: #78602d | Accent: #b49c67

Interactive Elements

Code Examples

Inline code like theme.generate() and larger blocks:

// Theme configuration
const theme = {
    name: "sharp_canyon",
    layout: "retro",
    colors: {
        primary: "#b98f33",
        secondary: "#78602d",
        accent: "#b49c67"
    },
    fonts: {
        heading: "roboto-slab",
        body: "oxygen",
        code: "intelone-mono"
    }
};

Content Structure

  • Clean, readable typography with 16px base font size
