Pure Storm

A wide layout with analogous colors

Welcome to Pure Storm

This theme features a carefully crafted wide layout with a dark analogous color scheme. The typography combines display fonts for headings with sans-serif for body text.

Typography Showcase

The Lato font family provides excellent readability for body text, while higher-jump adds character to headings. Code blocks use consola-mono for clarity.

Color Palette

Primary: #df3549 | Secondary: #d6753d | Accent: #d6218d

Interactive Elements

Links like this example and longer text links that demonstrate the theme's navigation style use the secondary color.

Code Examples

Inline code like theme.generate() and larger blocks:

// Theme configuration
const theme = {
    name: "pure_storm",
    layout: "wide",
    colors: {
        primary: "#df3549",
        secondary: "#d6753d",
        accent: "#d6218d"
    },
    fonts: {
        heading: "higher-jump",
        body: "Lato",
        code: "consola-mono"
    }
};

Content Structure

  • Clean, readable typography with 16px base font size
  • Heading scale ratio of 1.42x for visual hierarchy
  • Line height of 1.48 for comfortable reading
  • Wide layout optimized for content flow
  • Dark theme with analogous color harmony
